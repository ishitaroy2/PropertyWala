### Method 2: Setup Locally

Follow these steps to set up the application locally:


1. Deploying Frontend: <br/>
    - npm install <br/>
    - npm run dev  <br/>

1. Deploying Backend: <br/>
    - npm install <br/>
    - npm run dev  <br/>



The application will be up and running with frontend
